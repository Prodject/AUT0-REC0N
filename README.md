# AUTO-RECON
## Easy to use script to quickly scan a Target in Kali Linux

![alt text](https://github.com/gotr00t0day/AUT0-REC0N/blob/master/Auto-Recon2-7.PNG)


### INSTALLATION
```
cd /opt
git clone https://github.com/gotr00t0day/AUT0-REC0N.git
cd AUT0-REC0N
chmod +x setup.sh
./setup.sh
chmod +x recon.sh
```

### Usage:
```
./recon.sh
```
